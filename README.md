# 汪蕾个人网站

这是一个基于简历内容制作的单页个人作品集网站，适合继续扩展成求职主页或个人品牌站。

## 文件说明

- `index.html`：页面结构与文案内容
- `styles.css`：视觉样式、响应式布局与动效
- `main.js`：滚动 reveal 动画和页脚年份

## 本地预览

在当前目录运行：

```powershell
python -m http.server 8080
```

然后打开 <http://localhost:8080>。

## 后续建议

- 替换联系方式中的手机号展示策略，避免公开站点被采集
- 补充 GitHub、掘金、作品截图或线上项目链接
- 如果准备正式部署，可以再升级成 Vite 或 Next.js 版本
